# 정보처리기사 필기 모의고사

2024년 정보처리기사 필기 시험 대비 웹 기반 모의고사 애플리케이션입니다.

## 기능

- ✅ 100문제 랜덤 출제 (5과목 × 20문제)
- ✅ 100분 타이머 (일시정지/재개 가능)
- ✅ 실시간 답안 저장
- ✅ 과목별 채점
- ✅ 전체 시험 채점
- ✅ 상세 해설 제공
- ✅ 트리/그래프 시각화 (SVG)

## 사용 방법

1. "시험 시작" 버튼 클릭
2. 100문제 풀이 (타이머 확인)
3. "시험 종료" 또는 "현재 과목만 제출"
4. 결과 확인 (합격/불합격, 과목별 점수)

## 합격 기준

- 과목당 40점 이상 (과락 방지)
- 평균 60점 이상

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- SVG (트리 다이어그램)

## 라이선스

개인 학습용으로만 사용 가능합니다.
